# Color Tokens

The Lotus Warden color system is inspired by the lotus flower - emerging from dark depths into light. 
This symbolizes the security journey: finding vulnerabilities in the shadows and bringing clarity through illumination.

## Gradients

The design system includes several gradient presets:

- **Lotus Gradient**: `linear-gradient(135deg, #a855f7 0%, #e855a8 100%)`
- **Jade Gradient**: `linear-gradient(135deg, #22d3ce 0%, #3b82f6 100%)`
- **Gold Gradient**: `linear-gradient(135deg, #f0a500 0%, #ff6b6b 100%)`

## Usage

```tsx
import { primitives, semantic, darkTheme } from '@/design-system';

// Primitive colors
const purple = primitives.lotus[500]; // '#a855f7'

// Semantic colors
const criticalColor = semantic.severity.critical.base;

// Theme colors
const bgColor = darkTheme.bg.surface;
```
